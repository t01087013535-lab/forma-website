# 태동 메인 전시 랜딩 디자인 스펙

**날짜:** 2026-04-27  
**상태:** APPROVED  
**브랜드:** 태동 (Taedong)  
**목적:** 태동 메인 홈페이지 전체를 대체하는 브랜드 인지도 중심 랜딩 페이지

---

## 1. 목표

- 방문자가 첫 스크롤 10초 안에 태동을 일반 웹 에이전시가 아니라 `감각과 구조를 함께 다루는 팀`으로 기억하게 만든다.
- 전환 버튼보다 `장면 기억`을 우선한다.
- 태동의 실제 철학, 역량, 작업물을 고대 그리스 전시관 서사 안으로 번역한다.

## 2. 콘셉트

랜딩 전체는 `고대 그리스 전시관`을 현대적으로 재해석한 공간 경험이다.

- 사용자는 입구를 통과해 전시관에 `입장`한다.
- 스크롤은 단순 페이지 이동이 아니라 `다음 방으로 걸어가는 동선`처럼 느껴져야 한다.
- 초반 2개 방은 세계관 몰입형, 이후 3개 방은 실제 태동 역량과 컬렉션을 보여주는 콘텐츠형으로 전환한다.

## 3. 서사 구조

### 1막. Propylaea

- 석재 문이 열리며 브랜드 공간으로 진입하는 인트로
- 키 메시지: `고대 회랑을 지나 태동의 첫 장면으로.`
- 역할: 브랜드 몰입과 인상 각인

### 2막. Origin Hall

- 태동의 철학을 `전략, 설계, 구축, 운영` 네 개의 전시물로 배치
- 역할: 팀의 사고방식과 구조적 정체성 전달

### 3막. Atelier

- 브랜드 웹, 제품 구축, 시스템 설계, 운영 자동화를 전시품처럼 해석
- 역할: 실제 역량과 제작 풍경 제시

### 4막. Collection

- 대표 사례를 카드 그리드가 아니라 떠오르는 작품 패널처럼 노출
- 역할: 포트폴리오를 브랜드 경험의 일부로 통합

### 5막. Finale

- 개방된 돔 공간에서 브랜드 선언으로 마무리
- CTA는 조용하고 절제된 형태로 마지막에만 배치

## 4. 인터랙션 원칙

- `절충형` 인터랙션 강도를 유지한다.
- 인트로와 일부 핵심 구간은 강한 공간감을 주되, 전체를 무거운 3D 경험으로 만들지 않는다.
- 메인 구현은 `Lenis + Framer Motion` 기반의 2.5D 전개로 유지한다.
- 각 섹션은 `sticky viewport` 패턴을 사용해 같은 화면 안에서 공간이 전환되는 감각을 만든다.
- 모바일에서는 3D 강도를 줄이고 시각 서사와 타이포 위계를 유지한다.

## 5. 비주얼 시스템

- 색상: `obsidian black`, `marble sand`, `oxidized bronze`, `aegean blue`
- 소재감: 석재, 금속, 안개, 빛기둥, 먼지, 돔 공간
- 타이포: 대형 제목은 조형적 세리프 감각, 설명은 정제된 산세리프
- 인터페이스: 일반 SaaS 카드 UI 느낌 금지, 박물관 수준의 고요한 품격 유지

## 6. 구현 방향

- `app/page.tsx`는 새 랜딩 엔트리로 교체
- 데이터 구조는 `lib/taedong-exhibition.ts`로 분리
- 각 전시 방은 `components/landing/*` 아래 개별 컴포넌트로 분해
- 홈 전용 질감은 글로벌 CSS에 최소한만 추가
- Claude Design 전달용 프롬프트 파일을 별도 문서로 저장

## 7. 성능 및 접근성

- `prefers-reduced-motion` 환경에서는 모션을 크게 줄인다.
- 레이아웃 이동은 transform 기반으로 유지한다.
- 스크롤 전환은 영상 의존 대신 CSS/모션 기반으로 우선 구현한다.
- 전시실 이동은 앵커 네비게이션으로 접근 가능해야 한다.
